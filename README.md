# CITS4407 Assignment 2

## Overview

This project contains two Bash scripts for processing YouTube trending video data.

The `clean` script performs error checking and data cleaning on `trending_videos_unclean.csv`.

The `analyse` script performs data analysis on `trending_videos_clean.csv`.

## Files

- `clean`: Bash script for Question 1 data cleaning.
- `analyse`: Bash script for Question 2 data analysis.
- `README.md`: Explains how to run the scripts.
- `prompts.pdf`: Contains the AI prompt log.
- `git_backup`: Backup copy of the Git repository folder.

Run the cleaning script:

./clean trending_videos_unclean.csv
