# CITS4407 Assignment 2

This project contains two Bash scripts for data cleaning and data analysis.

## Files

- `clean`: cleans `trending_videos_unclean.csv`
- `analyse`: analyses `trending_videos_clean.csv`
- `README.md`: explains the project
- `prompts.pdf`: contains AI prompt log
- `git_backup`: backup copy of the Git history for submission

## How to run

```bash
./clean trending_videos_unclean.csv
./analyse trending_videos_clean.csv
